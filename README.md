# 后台管理系统

## 基于vue2+vuex+elementui+mockjs+echarts打造

## 项目初始化
```
npm install
```

### 项目调试
```
npm run serve
```

### 项目打包
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
